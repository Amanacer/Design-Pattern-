## 🔥 What's Inside?

> A curated collection of Java implementations of GOF (Gang of Four) design patterns, each with:
- 🔹 Simple and modular code
- 🔹 Real-world use-case analogy
- 🔹 Clean architecture separation

---

## 🧠 Patterns Included

### 🏗️ Creational
- ✅ Abstract Factory
- ✅ Factory Method
- ✅ Builder
- ✅ Singleton
- ✅ Prototype

### 🧱 Structural
- ✅ Adapter
- ✅ Decorator
- ✅ Proxy
- ✅ Composite
- ✅ Bridge

### 🔄 Behavioral
- ✅ Observer
- ✅ Strategy
- ✅ Command
- ✅ Iterator
- ✅ Chain of Responsibility
- ✅ Template Method

## ⚙️ Getting Started

### Prerequisites
- Java JDK 8 or higher
- Any Java IDE (Eclipse, IntelliJ, VS Code)

### Run Example
```bash
cd src/Builder
javac *.java
java Client
```
